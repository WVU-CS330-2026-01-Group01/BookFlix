# BookFlix Frontend

This folder contains the React + Vite frontend for BookFlix.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Notes

- The frontend expects the backend API at `VITE_API_BASE`, defaulting to `http://localhost:3000`.
- Authentication uses backend-issued HTTP-only cookies.
- Protected routes are handled in `src/App.jsx`.

For full project documentation, use the repo-level [docs hub](../../docs/README.md).
